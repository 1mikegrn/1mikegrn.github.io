pyGC

A desktop application for gas chromatography analysis

pyGC is a complete desktop application written in python and rendered in kivy. Users can import GC data sets and use pyGC to deconvolute n-number of GC distributions to determine the relative material quantity by integration.

pyGC can be installed with pip and git. Once installed, the application can be initialized in the python terminal by calling pyGC-init. Users are referred to the pyGC DocSite for more details.
